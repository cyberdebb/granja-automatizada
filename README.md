# granja-automatizada
Projeto final de Microprocessadores e Microcontroladores: uma granja automatizada em C

Para este projeto, foi utilizado o software Proteus 8 Professional e o MPLABX IDE v5.35
- O repositório contém os arquivos de simulação no Proteus, os arquivos de source em C e o esquema da placa de circuito impresso (pcb)
